# WageWatch India

AI-powered wage inequality detection and labor law violation analysis platform using Classical Machine Learning and Quantum Machine Learning.

## Overview

WageWatch India analyzes wage data across multiple Indian states to identify:
- gender-based wage gaps
- labor law violations
- underpayment patterns
- enforcement severity
- policy intervention priorities

The platform combines machine learning, data analytics, and quantum-enhanced models to support data-driven labor policy decisions.


## Features

- Wage inequality detection
- Gender wage gap analysis
- Labor law violation prediction
- Severity scoring system
- State-wise enforcement analysis
- Interactive dashboards and visualizations
- Classical ML + Quantum ML comparison
- Policy recommendation engine
